### 🎯 模板标准命名

定制化双边无向弦图 / 带自定义图例的关系映射弦图 (Customized Bipartite Undirected Chord Diagram / Relational Chord Chart with Custom Legend)

### 🏷️ AI 检索高频关键词 (Tags)

建议将这些中英双语的专业术语作为标签保存，精准覆盖它的拓扑结构与高阶定制特征：

- **基础图形** : 弦图 (Chord diagram), 无向弦图 (Undirected chord diagram), 双边弦图 (Bipartite chord diagram/plot).
- **布局与拓扑** : 圆形布局 (Circular layout), 双分类网络映射 (Two-category network mapping), 多对多互作关系 (Many-to-many interaction).
- **视觉细节** : 线性刻度 (Linear ticks), 局部标签隐藏 (Selective label hiding/masking), 手动图例构建 (Manual patch legend), 半透明弦带 (Translucent chords), 节点独立着色 (Independent node coloring).
- **适用场景** : 物种共生/互作分析 (Species interaction/symbiosis), 微生物群落物质转移 (Microbial community transfer), 复杂共现网络 (Co-occurrence networks).
