### 🎯 模板标准命名

**环形聚类热图 / 极坐标复合热图 (Circular Clustered Heatmap / Radial Clustered Heatmap)**

### 🏷️ AI 检索高频关键词 (Tags)

建议将这些中英双语的高频词作为标签保存，涵盖了该图表的几何特征、组成部分和常见应用：

- **基础图形**: 环形热图, 极坐标热图, 环状相关性图, circular heatmap, radial heatmap, annular heatmap.
- **边缘组件**: 极坐标树状图, 环形层次聚类, 径向树图, radial dendrogram, circular hierarchical clustering.
- **视觉细节**: 径向旋转文本 (radial text labels), 扇形网格填充 (polygon fill/wedge grid), 连续型颜色条 (continuous colorbar), 极坐标转换 (polar transformation).
- **适用场景**: 多组学数据可视化 (multi-omics visualization), 微生物群落分析, 复杂特征相关性展示.

### 🗂️ 结构化特征解析 (模板属性卡片)

| **核心组件**       | **绘图类型与视觉特征**                                       |
| ------------------ | ------------------------------------------------------------ |
| **整体坐标系**     | **无边框极坐标系隐喻**。底层基于笛卡尔坐标系（直角坐标），通过三角函数（sin/cos）和极坐标转换，将直线和矩形硬编码映射为圆弧和扇形。隐藏了所有 X/Y 轴和网格。 |
| **中心与边缘节点** | **极坐标聚类树 (Radial Dendrogram)**。将传统 `dendrogram` 的直线分支通过弧度插值弯曲。内侧树状图由圆心向外发散，侧边树状图贴合外环。 |
| **主体视觉**       | **环形热图 (Annular Heatmap)**。不使用常规的 `imagesc`，而是利用多边形填充（`fill`）循环绘制每一个扇形网格单元，并根据自定义的连续色带（Colormap）映射颜色。 |
| **标签与图例**     | **径向文本与独立色条**。数据标签围绕外环分布，文本的旋转角度（Rotation）随着所在位置的极角动态计算，保证呈放射状易于阅读；侧边配有标准连续型 Colorbar。 |
