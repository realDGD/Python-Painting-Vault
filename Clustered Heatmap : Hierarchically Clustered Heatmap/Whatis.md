### 模板标准命名

**聚类热图 / 带聚类树的复合热图 (Clustered Heatmap / Hierarchically Clustered Heatmap)**

### 🏷️ AI 检索高频关键词 (Tags)

建议将这些中英双语的专业术语作为标签保存，涵盖了该图表的所有核心组件和应用场景：

- **基础图形:** 聚类热图, 相关性热图, 矩阵热图, clustered heatmap, correlation matrix heatmap.
- **边缘组件:** 层次聚类, 系统聚类树, 树状图, hierarchical clustering, dendrogram, marginal plots.
- **视觉细节:** 连续型颜色条 (colorbar), 发散型配色 (diverging colormap), 白色网格分隔线 (white grid lines), 行列重排 (row/column reordering).
- **适用场景:** 基因表达分析 (gene expression), 组学数据, 特征相关性分析 (feature correlation).

### 🗂️ 结构化特征解析 (便于归档)

| 核心组件                    | 绘图类型与视觉特征                                           |
| --------------------------- | ------------------------------------------------------------ |
| **主坐标系 (Center)**       | **矩阵热图 (Heatmap)**。特征：基于数据值映射颜色；利用循环绘制了白色的横纵网格线进行单元格分隔；X/Y轴标签根据聚类顺序进行了重排 (reordering)；Y轴标签放置在右侧 (right Y-axis)。 |
| **边缘坐标系 (Top & Left)** | **层次聚类树 (Dendrogram)**。特征：利用绝对位置比例紧密贴合主热图；去除了背景、坐标轴线和刻度，仅保留黑色的树状分支结构。 |
| **图例设计 (Right)**        | **连续型颜色条 (Colorbar)**。特征：置于画布右侧外缘，使用插值 (interpolation) 生成了平滑的发散型渐变色图 (Colormap)。 |
| **数据逻辑**                | 原始矩阵数据必须经过 `linkage` (计算距离) 和 `dendrogram` (生成树并获取重排索引) 处理后，再用新的索引重组矩阵进行热图绘制。 |
