### 🎯 模板标准命名

旋转三角形聚类热图 / 45度对角相关性热图 (Rotated Triangular Clustered Heatmap / Diagonal Correlation Heatmap)

### 🏷️ AI 检索高频关键词 (Tags)

建议将这些中英双语的专业术语作为标签保存，精准覆盖它的几何变换和空间压缩特征：

- **基础图形:** 三角形热图, 聚类相关性矩阵, 金字塔热图, triangular heatmap, diagonal correlation matrix, pyramid plot.
- **布局与拓扑:** 45度旋转 (45-degree rotation), 矩阵半角/下三角 (half matrix / lower triangular), 几何坐标系转换 (geometric coordinate transformation).
- **视觉细节:** 菱形网格填充 (diamond grid / polygon fill), 斜向对齐树状图 (diagonal-aligned dendrogram), 斜交文本标签 (slanted text labels).
- **适用场景:** 组学对称数据（如相关系数）的空间压缩展示, 复杂变量共线性分析可视化.

### 🗂️ 结构化特征解析 (模板属性卡片)

| **核心组件**                | **绘图类型与视觉特征**                                       |
| --------------------------- | ------------------------------------------------------------ |
| **整体坐标系**              | **无边框旋转坐标系**。隐藏了常规的 X/Y 轴边框与背景。底层仍然是笛卡尔坐标系，但所有视觉元素的坐标计算都经过了特定的平移和 45 度旋转隐喻，呈现悬浮的金字塔或金刚石形态。 |
| **主体视觉 (Heatmap)**      | **旋转多边形网格**。放弃了原生的图像映射函数（如 `imagesc`），利用 `fill` 函数循环绘制闭合多边形，使得方形单元格在视觉上转换为紧凑的菱形阵列。 |
| **边缘树状图 (Dendrogram)** | **斜向贴合聚类树**。提取标准层次聚类树的线条原始坐标数据（XData/YData）后，套用旋转矩阵公式（如 `nX = tX*cos(pi/4) - tY*sin(pi/4)`）进行二次重绘，使其完美贴合倾斜的主图。 |
| **标签与图例 (Labels)**     | **斜交文本与独立色标**。文本标签的坐标系经过人为偏移计算，并强制附加了旋转属性（Rotation=45），使其与几何网格的走向保持平行；底部留白放置标准 Colorbar。 |
